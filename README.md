# Marketing Campaign Analytics Dashboard

## Description
This project analyzes customer purchasing behavior and marketing campaign performance using Python, MySQL, Pandas, Plotly, and Streamlit.

## Features
- Dashboard
- Product Analysis
- Campaign Analysis
- Customer Segmentation
- Channel Analysis
- Customer Insights
- SQL Analytics

## Technologies
- Python
- Streamlit
- MySQL
- Pandas
- Plotly Express

## Run the Project

Install dependencies:

pip install -r requirements.txt

Run:

streamlit run app.py
